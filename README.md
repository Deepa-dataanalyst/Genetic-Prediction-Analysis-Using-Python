# Genetic-Prediction-Analysis-Using-Python
## Project Overview
This project generates and analyzes a dataset of 1,000 genetic testing kits 
using Python and Pandas.

The Objective is to identify restricted genetic prediction kits and perform 
category and pricing analysis.
## Technologies Used
* Python
* Pandas
* Jupyter Notebooks
## Dataset Columns
* Prediction_Tool_ID
* Title_Prediction
* Category
* Price
* Restricted_Status
  ## Analysis Performed
  1. Total number of Prediction Kits
  2. Restricted vs Non Restricted kit count
  3. Percentage of restricted kits
  4. Category distribution analysis
  5. Average price by category
  6. Most expensive prediction kit
  7. Cheapest kit
## Key Learning Outcomes
* Data generation using Python
* Creating DataFrames with Pandas
* Data Filtering and analysis
